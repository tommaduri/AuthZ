## Chart Types

Criterion generates multiple chart types for each benchmark:

- **typical.svg**: Representative iteration time
- **mean.svg**: Mean performance with confidence intervals
- **median.svg**: Median performance (less affected by outliers)
- **pdf.svg**: Probability density function
- **regression.svg**: Linear regression analysis
- **violin.svg**: Distribution visualization

## Regenerating Charts

To regenerate all charts with updated benchmarks:

```bash
# Run benchmarks (generates charts automatically)
cargo bench --workspace

# Save new baseline
cargo bench --workspace -- --save-baseline phase-5-updated

# Copy charts to docs
cp target/criterion/consensus_engine/single_vertex/report/*.svg docs/benchmarks/charts/
```
